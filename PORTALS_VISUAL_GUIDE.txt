┌─────────────────────────────────────────────────────────────────────────────┐
│                   CITIZEN JOURNEY (Step by Step)                             │
└─────────────────────────────────────────────────────────────────────────────┘

STEP 1: ENTRANCE (Kiosk Portal)
┌─────────────────────┐
│   Citizen arrives   │
│   Uses touchscreen  │
│   Enters ID & name  │
│   Selects service   │
│   Gets ticket       │
│   IM-016            │
└──────────┬──────────┘
           │
           ▼
STEP 2: WAITING AREA (Display Portal)
┌─────────────────────┐
│   Sits and waits    │
│   Watches TV screen │
│   Sees IM-016 in    │
│   waiting queue     │
└──────────┬──────────┘
           │
           ▼
STEP 3: CALLED (Display Portal)
┌─────────────────────┐
│  TV shows:          │
│  NOW SERVING        │
│  IM-016             │
│  COUNTER 1          │
└──────────┬──────────┘
           │
           ▼
STEP 4: AT COUNTER (Counter Portal)
┌─────────────────────┐
│  Goes to Counter 1  │
│  Staff verifies ID  │
│  Service provided   │
│  Staff marks done   │
└──────────┬──────────┘
           │
           ▼
STEP 5: COMPLETE!
┌─────────────────────┐
│  Service finished   │
│  Citizen leaves     │
│  Stats updated      │
└─────────────────────┘

TO OPEN PORTALS:
  Double-click the HTML files or use these URLs:

  1. file:///D:/Queue%20Management%20Standard/kiosk_portal.html
  2. file:///D:/Queue%20Management%20Standard/counter_portal.html
  3. file:///D:/Queue%20Management%20Standard/display_portal.html
  4. file:///D:/Queue%20Management%20Standard/demo_dashboard.html

TO TEST:
  1. Open all 4 in separate windows
  2. Create ticket in Kiosk
  3. Watch Display update
  4. Call from Counter
  5. See Display show counter number
  6. Monitor in Demo Dashboard

TO SETUP MULTIPLE COUNTERS:
  1. Copy counter_portal.html
  2. Edit: const COUNTER_ID = 2 (or 3, 4...)
  3. Open on different computer
  4. Display shows all counters!

SUMMARY:
  4 Portals = 4 Jobs
  1. Kiosk = Get Ticket (Citizens)
  2. Counter = Serve Customer (Staff)
  3. Display = Watch Queue (Everyone)
  4. Demo = Manage System (Admin)

  All work together in real-time!
  Display now shows ALL counters (fixed!)
